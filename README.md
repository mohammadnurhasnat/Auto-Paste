# Auto-Paste

এটি যেভাবে আপনাকে অন/অফ সিগন্যাল দেবে:
মোড চালু হলে (ON): যখনই আপনি বুকমার্কে ক্লিক করবেন, স্ক্রিনের ডানদিকের ওপরের কোণায় একটি সুন্দর সবুজ রঙের ক্যাপসুল আকৃতির ব্যাজ ভেসে উঠবে যেখানে লেখা থাকবে Auto-Paste: ON। এটি দেখে আপনি নিশ্চিত হতে পারবেন যে এখন ক্লিক করলেই পেস্ট হবে।

মোড বন্ধ হলে (OFF): কাজ শেষে বুকমার্কে আবার ক্লিক করামাত্রই সেই সবুজ ব্যাজটি স্ক্রিন থেকে একদম ভ্যানিশ হয়ে যাবে। ফলে আপনি বুঝবেন মোডটি বন্ধ হয়ে গেছে এবং আপনার মাউস আবার সাধারণ অবস্থায় ফিরে এসেছে।
